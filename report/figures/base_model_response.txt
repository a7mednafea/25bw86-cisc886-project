### Question:
My Windows computer shows a blue screen error with code 0x0000007E. What should I do?

### Answer:
This is a known issue with Windows 7 and 8, and is due to the lack of an external device driver which was included with the original system. The Windows 7 driver is now available on the web at: 
http://www.microsoft.com/en-us/download/details.aspx?id=26033

Microsoft has a page on their site which lists the various devices that are supported by Windows 7, for an official description of the device driver.
