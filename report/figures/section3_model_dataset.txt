Last Updated: March 2025

Description:
This dataset contains 30.4 million question-and-answer pairs extracted
from the entire Stack Exchange network — the world's largest tech support
and IT community. It includes posts from SuperUser (computer hardware and
software), AskUbuntu (Linux support), ServerFault (server administration),
and hundreds of other technology-focused communities. Each record in the
'text' field contains a real question asked by a user followed by the
accepted or highest-voted answer. This makes it the ideal dataset for
training a Tech Support Chatbot, as it contains authentic user problems
and expert solutions covering operating systems, networking, hardware,
software errors, cybersecurity, and cloud computing.

Why this dataset fits the Tech Support domain:
- SuperUser.com: Windows/Mac/Linux hardware and software troubleshooting
- AskUbuntu.com: Linux and Ubuntu specific support
- ServerFault.com: Server and network administration
- All posts are real user questions with verified expert answers

Split Strategy:
- Training set:   80% (~24.3 million records)
- Validation set: 10% (~3.04 million records)
- Test set:       10% (~3.04 million records)
- Method: PySpark randomSplit() with fixed seed=42
- Data leakage prevention: split by record ID so no question
  appears in more than one split
